# **说明**

+ ### 这是一个方便快速将多张图片自动导出为Mipmap的超小型工具
+ ### 将你准备好的图片命名为“p0.png”、“p1.png”、“p2.png”等格式
+ ### 将所有命名好的图片放入“Input”文件夹
+ ### 点击运行“AMIP.exe”
+ ### 在“Output”获取你的导出文件
+ ### 运行成功后按“Enter”关闭cmd窗口

# **注意事项**
+ ### 请严格遵守命名规则
+ ### 请确保电脑留有512mb的运行内存和1gb存储空间以保证运行成功
